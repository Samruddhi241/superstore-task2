📘 README – Task 2: Data Visualization & Storytelling

1. Project Title

Sales Data Visualization & Storytelling using Power BI / Tableau


---

2. Objective

The goal of this task is to create meaningful and interactive data visualizations that help understand business performance.
The focus is not only on charts, but on telling a clear story using insights from the data.


---

3. Dataset Used

Superstore.csv
(This dataset contains information about sales, profit, categories, customers, and regions.)


---

4. Tools Used

Power BI / Tableau (any one)

Optional: Excel for quick checking



---

5. Steps Performed

5.1 Data Import

Loaded Superstore.csv into Power BI/Tableau.

Checked column names and data types.


5.2 Data Cleaning (Minor)

Removed blank rows/records.

Ensured date column was in correct datetime format.

Standardized category/segment names if required.



---

6. Visualizations Created

1️⃣ Sales by Category – Bar Chart

Shows which product category performs best.

Highlight: Technology has highest sales.


2️⃣ Profit by Sub-Category – Tree Map / Bar Chart

Identifies loss-making products.

Insight: Tables and Bookcases show negative profit.


3️⃣ Sales by Region – Map Visualization

Shows performance across regions.

Insight: West region performs the strongest.


4️⃣ Monthly Sales Trend – Line Chart

Shows how sales changed over months/years.

Insight: Sales peak during year-end season.


5️⃣ Customer Segment Contribution – Donut/Pie Chart

Shows contribution of Consumer, Corporate, Home Office.

Insight: Consumer segment contributes the largest share.


6️⃣ Top 10 Products by Sales – Bar Chart

Highlights best-selling products.



---

7. Dashboard Design Guidelines Followed

Used minimal colors (2–3) for clean look.

Avoided clutter and unnecessary elements.

Added labels, titles, and short insights on each chart.

Maintained proper alignment for readability.



---

8. Insights / Story (Summary)

Technology category drives the highest sales.

Tables and Bookcases generate losses → need price improvement.

West region leads in sales performance.

Seasonal spike observed in November–December.

Consumer segment is the largest contributor to revenue.


These insights help in better marketing planning, pricing strategy, and inventory management.
